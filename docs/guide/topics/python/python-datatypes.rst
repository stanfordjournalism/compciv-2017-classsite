Integers
========

Your standard whole numbers.

.. code-block:: python

    >>> 2 + 3

Floats
======

Numbers with decimal places.

.. code-block:: python

    >>> 2.2 + 3.3
    5.5


Integers and floats can be generally used interchangeably without causing an error:


.. code-block:: python

    >>> 2 * 0.5
    1.0
