*******************************************************
find - find deeply nested files and directories by name
*******************************************************





Examples
========


****************************
Search for directory by name
****************************


In my `/tmp` directory, find all directories that are named `.git`

.. code-block:: shell

    $ find /tmp -type d -name '.git'
